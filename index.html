ChemCam

ChemCam is a website which provides users with the ability to input a formula name, either through text or an image, and be displayed with the chemical formula. ChemCam supports single element, binary acid, oxyacid, covalent bond, compound, multivalent compound, acid radical, and multivalent acid radical conversions.
